- HTTP/0.9
  - 1991년 버전
  - 오직 GET메서드만 지원
  - 간단한 HTML객체를 받기 위해 만들어짐
- HTTP/1.0
  - HTTP헤더, 추가 메서드, 멀티미디어 객체 처리 추가
  - 웹페이이지 폼 실현
- HTTP/1.0+
  - 1996년 중반
  - 다양한 웹클라이언트와 서버로 인해 규격 외 확장된 HTTP 기능 추가
  - ‘keep-alive’커넥션, 가상 호스팅, 프락시 연결 지원 등
- HTTP/1.1
  - 1997년 등장
  - 구조적 결함 교정, 성능 최적화, 잘못된 기능 제거
- HTTP/2.0
  - 구글의 SPDY프로토콜을 기반으로 설계 진행중인 프로토콜
